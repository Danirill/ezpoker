# EZ Poker

Локальный веб-покер **Texas Hold'em** для игры в браузере против ботов.

![Стек](https://img.shields.io/badge/React-19-blue) ![TypeScript](https://img.shields.io/badge/TypeScript-5.8-blue) ![Vite](https://img.shields.io/badge/Vite-6-purple)

## Быстрый старт

```bash
npm install
npm run dev
```

Откройте [http://localhost:5173](http://localhost:5173) в браузере.

## Сборка

```bash
npm run build
npm run preview
```

## Как играть

1. Нажмите **«Начать игру»** / **«Новая раздача»**
2. Вы играете за столом с **4 ботами** (всего 5 игроков)
3. Стартовый стек: **2000 фишек**, блайнды **10/20**
4. Доступные действия: **Фолд**, **Чек**, **Колл**, **Рейз** (с ползунком), **Олл-ин**
5. Боты ходят автоматически с небольшой задержкой

## Архитектура

```
src/
├── game/           # Игровая логика (без UI)
│   ├── pokerEngine.ts   # Состояние, раздача, ставки, шоудаун
│   ├── handEvaluator.ts # Оценка комбинаций Texas Hold'em
│   ├── bot.ts           # AI ботов
│   ├── deck.ts          # Колода и перемешивание
│   └── types.ts         # Типы
├── components/     # React UI
│   ├── PokerTable.tsx   # Стол и раскладка мест
│   ├── Card.tsx         # Карта с flip-анимацией
│   ├── Chips.tsx        # Стопка фишек
│   └── ActionPanel.tsx  # Панель действий
└── hooks/
    └── usePokerGame.ts  # Связка UI ↔ движок + ходы ботов
```

**Стек:** Vite + React + TypeScript. Сервер не нужен — вся логика на клиенте, Vite отдаёт статику в dev-режиме.

## Анимации

- **Раздача карт** — полёт сверху с масштабированием (`deal-card`)
- **Переворот карт** — 3D flip (`rotateY`) для закрытых карт ботов
- **Фишки** — стопка с градиентом, пульсация при пополнении банка
- **Ставки** — плавное появление у мест игроков
- **Активный игрок** — подсветка рамкой
- **Панель действий** — slide-in при вашем ходе
